# Muffin Text 0.2 Alpha

## Scripts
- Apenas um script
- Sem distinção real entre local/global (globais apenas)

## Variáveis
- Tipos suportados: número, string
- Comando: Variablet (set, change) (global, local) (nome) (valor)
- Sobrescrita permitida

## Comandos disponíveis
- Variablet (set, change) (global, local) (nome) (valor)
- Print (coisa)
- Repeat
- ###If (variável) (=, <, >) (valor)

(comandos entre ###if e ###endif)

###endif

- Setflag (nome) (vezes em que pode ser chamada)
quando uma flag é chamada, o executor vai até a flag e continua a partir dela, se a flag estiver inválida, nada será feito.
- Gotoflag (nome da flag)
- setrule (regra) (valor)

## Regras do Executor
- **Loudscript:** se definido para true, retorna mai informações sobre a execução, útil para debug.
- só isso kk

## If e comparação
- Comparação direta
- Uso de parênteses para variáveis
- Limitações conhecidas

## Executor
- Modelo parser + executor
- Execução sequencial
- FPS Baixo Fixo

## Limitações Conhecidas
- FPS Capado a 32
- Sem suporte a vários scripts
- Sem suporte a comentários
- Sem suporte a salvamento de projetos
